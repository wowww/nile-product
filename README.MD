## 시작하기

### 패키지 설치

```bash
yarn install
```

## 스크립트

### 스토리보드 실행(로컬 개발 환경)

- http://localhost:6006

```bash
yarn storybook
```

### 서비스 실행(로컬 개발 환경)

- http://localhost:3000

```bash
yarn dev
```

### 빌드(배포 환경)

```bash
yarn build
```

### 서버 시작(배포 환경)

```bash
yarn start
```

### 이미지 업로드
```bash
yarn upload:beta
```

### 이미지 업로드(assets)
```bash
yarn upload-assets:beta
```

### 이미지 업로드(public)
```bash
yarn upload-public:beta
```
